This is second generation of Timekeeper application. Main change in on technology used to create this application and concept to bring more efficiency and flexibility on time management. You can download application for windows hier. If you need version for macOS or linux let me know rzi@vp.pl



How install it

After downoad you can click on the Timekeeeper2.0.0.exe file and automatic installer will follow you via installation process. When installation process will finish you will see Timekeeper icon on your desktop and application will be open, ready to configuration and use. Easy? easy!

How to use it - general

When application is open, you can find on your screen two windows. First window is for meeting animator, to change settings and operate during presentation of presenters. On first line you can see current date and time. Next line show name of conference and sheduled meeting time. how to setup ptesenters and time you can see on chapter settings. On third line we can write message what could be wisible for presenter and audiences on second window. Last part is for list of presenters order by sequence of presenter.

Second window is to vizualize progress of presentation for presenters and also for audiences. Presenter and also animator can click button next when first presenter finish his/her presentation

First of all - settings

If you would like to change settings, you should navigate to hamburger menu on right top corner and click settings in left side menu list. You will see settings page divided on two sections. On top section you can add name of meeting and new presenters with alocated time for their presentation. Second bottome section is for editing name and time already added presenters and deleted presenters if we dont need it.

Below you can see screen setting page when you would like to edit settings. First, click on the line that you woiuld like to edit an next click edit button.

Right now you can modifie time and presenter name. If you did it, you can save change by click buton save.


If you would like to deletew presenter from presenters list, mark which line you would like to delete and click delete buton. Marked lina is indicated on yelloe color

We have application ready to use, Let say thet meetinh name is test and we have three presenters (presenter1 , presenter2, presenter3) with alocated time 1 minute for presenter1, 2 minutes for presenter2 and 3 minutes for presenter3, so we are ready to start our meeting.

To start presenter we click buton play on main page. Color of buton will change from balck to red it means presenter is duing his/her presentation.

When presentation of current presenter is finish, animator click buton next on window progress bar and we give floor to next presenter. The same functionality is if we click next play button on main page. After finish first presentation on main page we see result of first presenter, it means his/her result in procent of used time and used time in minuses. See details on screen below

We repate the same for each presenter until all play butons are green. Gren buttons means that presentations are finished
