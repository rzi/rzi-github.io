Timekeeper 1.0

Introduction

This application was prepared for managers to supporting them during managment meeting for time management. There is sequence of module presented by managers with dedicated time. Role of timekeeper is to annimate and introduce each presenters and summarize results and the and of meeting.



How does it works

Before meeting, each presenters agred time for presentation with meeting animator and animator setup time in time input field. When the meeting start animator klick start button and give flor to each preseter accirding to defined sequence of the meeting.


How to use it

This is simple, animator start each presetners by click start buton for presener who wil start his presentation. On separated window all audience can see progres of prsesntation in reference to setup time. this window can be located on the presenter screen,on projector. On main window you will see result of each finished presentations which display time os begin of presentation, time of end of presentation and result, it means difference between start and stop of each presetations.


How check records from database

This application is connected to external mySQL database where all records are store. On the tab history (historia) you can see all registered records. You can't modify this record, this is only view.


How to show chart

For results visualization you can use waterrfall chart on the tab Charts (wykresy). On the vertical bars there is value of result presenter time in in order of presentarions. Last bar show total tim of meeting. Red colos means that presenter over the time. Green colos show that pesenter was on time or finish his/her presentation before schedule.


How to setup connection with database

On the setting tab you can decide to store yous records in local storage or on external mySQL server. After set checkbox, you can put connection data to your data base.
